# วิธี run

## สร้าง .env ใน back-end โดยมี DB คือ setting database ตัวอย่าง `user:pass@tcp(127.0.0.1:3306)/dbname?charset=utf8mb4&parseTime=True&loc=Local` และ SECERET คือ jwt signature

#### back-end
1`cd back-end`\
2`go run main.go`

#### font-end
1.`cd front-end`\
2.`npm install`\
3`npm start`\
4`จะเปิดที่ localhost:3000`
